# Happy Women's Day, Francesca! 💐

A personalized, interactive International Women's Day 2026 celebration site.

## Deploy to GitHub Pages

1. Create a new repository on GitHub (e.g., `womensday-francesca`)
2. Push this folder to the repo:
   ```bash
   git init
   git add .
   git commit -m "Happy Women's Day!"
   git branch -M main
   git remote add origin https://github.com/YOUR_USERNAME/womensday-francesca.git
   git push -u origin main
   ```
3. Go to **Settings → Pages** in your repo
4. Under "Source", select **Deploy from a branch**
5. Choose the `main` branch and `/ (root)` folder
6. Save — your site will be live at `https://YOUR_USERNAME.github.io/womensday-francesca/`
